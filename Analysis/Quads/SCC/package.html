An implementation of Sparse Conditional Constant propagation for SSI form, extended to do predicated optimizations. Removes many null-pointer and array-bounds checks inserted due to Java semantics.

Related Documentation

For further information, see:
  • Mark N. Wegman and F. Kenneth Zadeck, "Constant propagation with conditional branches", ACM Transactions on Programming Languages and Systems, Vol. 13, No. 2 (April 1991), Pages 181-210.
  • C. Scott Ananian, "Static Single Information Form", MIT Master's Thesis. Includes a section on the modified SCC implemented here.
