Useful utility classes, including general-purpose algorithms and data-types as well as FLEX-specific utility functions.

Of interest are:

  • Various classes to synthesize new Iterators from existing Iterators or Enumerations.
  • Useful datatypes: Tuples, WorkSets, PersistentMaps, and heaps.
  • Utility functions to make using the Collections API less painful: AbstractMapEntry, Default, UnmodifiableIterator, ListComparator.
  • Useful functions, such as assert, escape, grow, and shrink, in harpoon.Util.Util.
  • Functions for extracting features of HClasses, in harpoon.Util.HClassUtil.
