A flat, typed, high-level quadruple-based IR.

Quad form is used for most analysis and transformation in the compiler. It maintains strong typing, and tries to be minimal. There are three main variants:

  • QuadWithTry, which contains HANDLER quads to handle thrown exceptions, and does implicit safety-checking.
  • QuadNoSSA which handles all exceptions explicitly.
  • QuadSSA which is the same as the above, except it is in SSI form.

These varieties are somewhat misnamed; hopefully this will be fixed soon. For lower-level transformations involving explicit pointer arithmetic, LowQuad form must be used.
